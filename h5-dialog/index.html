移动端弹层组件

SDK 接入指南（JS）

海边业务服务平台 提供 JS SDK,将代码插入到产品线的网页中以后就可以呼出快速登录组件。

整合JS SDK至web 环境

将我们提供给您的 JS SDK 加入到您所需要弹层登录的页面，请将我们给您提供的 JS SDK 复制到 <head> 和 </head> 标签之间即可, 例如：

<script type='text/javascript'>
    var _hbmp_passport = _hbmp_passport || [];
    window._hbmp_passport = _hbmp_passport;
    (function(){
        var _app_ = document.createElement("div"); 
        _app_.setAttribute('id','fastlogin');
        document.body.appendChild(_app_); 
        _hbmp_passport.push(['product', '业务线标示']);
        ;(function() {
            var _passport_ = document.createElement('script');
            _passport_.type='text/javascript';
            _passport_.async = true;
            _passport_.src = '//mpres.haibian.com/fastlogin/static/js/fastlogin.js';
            var s = document.getElementsByTagName('script')[0];
            s.parentNode.appendChild(_passport_, s);
        })();
    })();
</script>

成功加载SDK后弹层组件会自动创建窗体内容、登录表单、异步请求等。具体设置方法请参考下面的详细说明。

用户自定义维度

服务平台本身提供了例如 “弹窗标题”，“弹窗描述”，“登陆后的跳转地址”,“登录按钮样式”,”按钮文字”,”是否禁用”,”按钮是否为镂空”,”按钮是否为圆角” 等等这些维度：

    _hbmp_passport.push(['title', '弹窗标题']);
    _hbmp_passport.push(['desc', '弹窗描述']);
    ...
    _hbmp_passport.push(['btnType', 'info']);

页面属性

当开发者接入完成JS SDK 后，还需要做最后一步。在需要调出弹层的连接或者按钮上加上一个ID标示。例如：

<a href="javascript:;" id="hbFastLogin">登录</a>
